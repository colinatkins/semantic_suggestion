How Quality Settings Work
Simple Rule: Scheduler decides what to save, TypoScript decides what to show
Storage (Scheduler)

Decides which page pairs are worth saving for later display.

Example: Setting = 0.3
Saves pairs with score: ≥ 0.3 (direct mapping)
Higher = fewer pairs stored, faster analysis
Display (TypoScript)

Controls what suggestions users actually see on the frontend.

Example: Setting = 0.4
Shows: Pairs with score ≥ 0.4
Higher = fewer but better suggestions

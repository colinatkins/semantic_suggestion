Configuration requise :

  1. Allez dans le module Système > Scheduler
  2. Créez une nouvelle tâche "Semantic Suggestion: Generate Similarities"
  3. Configurez les paramètres :
    • Start Page ID : ID de la page racine d'analyse (ex: 1)
    • Minimum similarity threshold : Seuil de stockage (ex: 0.3)
    • Pages to exclude : Pages à exclure (optionnel)
  4. Programmez l'exécution (quotidienne/hebdomadaire en heures creuses)
  5. Exécutez la tâche une première fois
Note : La première exécution peut prendre du temps selon le nombre de pages à analyser.

Sélection de l'analyse

Chaque analyse correspond à une exécution de la tâche Scheduler avec un Start Page ID spécifique.
